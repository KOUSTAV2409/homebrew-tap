# homebrew-tap

Personal Homebrew tap for installing [nf (Note Fast)](https://github.com/KOUSTAV2409/nf).

## Installation

To add this tap and install `nf`, run:

```bash
brew install KOUSTAV2409/tap/nf
```

Or tap the repository first and install:

```bash
brew tap KOUSTAV2409/tap
brew install nf
```
